# AlgoByte Website

This is the official website for AlgoByte, an AI solutions company.

## Structure

- `src/`: Contains all source files
  - `pages/`: HTML files
  - `styles/`: CSS files
  - `assets/`: Images and fonts

## Updating the Site

1. Edit HTML in `src/pages/index.html`
2. Modify styles in `src/styles/` if needed
